# Quartz Watch PCB Inspection

This project documents a university course project involving a quartz watch and a PCB inspection task.

## Context

As part of a university course, a quartz watch was built. During the course, an additional task was to inspect another PCB and find why it did not work as expected.

## Problem

The PCB did not work correctly. A short visual inspection did not show a clear defect.

## Method

The board was checked visually first. Because the suspected issue could not be confirmed from the outside in the available time, an X-ray device was used for non-destructive inspection.

## Result

The final cause was an incorrectly mounted component. The component was mounted upside down, so the circuit could not work correctly.

The finished quartz watch was later given to me during the exam.

## What this project shows

This project shows a systematic inspection approach:

- Start with a visual inspection
- Form a hypothesis
- Use non-destructive inspection when needed
- Compare suspicion and evidence
- Identify the real assembly issue
