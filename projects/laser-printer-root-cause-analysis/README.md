# Laser Printer Root Cause Analysis

This project documents the troubleshooting of a laser printer with an unusual print image.

## Problem

The printer produced an incorrect print image.

## Goal

The goal was to identify the root cause through testing instead of replacing parts without evidence.

## Method

The troubleshooting process was done step by step.

Sensors and the mechanical sequence were checked to understand where the printing process did not work correctly.

The issue was traced back to a mechanical timing problem: a gear was rotated incorrectly.

## Result

The root cause was identified through systematic testing instead of guessing.

## What this project shows

This project shows a practical root cause analysis approach:

- Observe the print pattern
- Check possible causes step by step
- Test sensors and mechanical behavior
- Follow the process sequence
- Identify the real root cause
