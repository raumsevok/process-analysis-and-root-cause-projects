# USB Trigger Charging Check

This project documents a practical USB power diagnosis using a USB trigger and a USB meter.

## Problem

A small electronic device did not behave as expected during charging.

## Goal

The goal was to check whether the device showed measurable power draw.

## Method

A USB trigger and a USB meter were used to observe the electrical input values during the test.

## Result

The device showed power draw, but it still did not recover. No deeper repair attempt was continued at that time.

## What this project shows

This project shows a practical diagnostic approach:

- Define a test condition
- Observe measured values
- Compare expectation and result
- Decide whether further troubleshooting is useful
