# Process Analysis and Root Cause Projects

This repository documents hands-on technical projects focused on systematic problem analysis, measurement-based investigation, root cause finding, and process improvement.

## Purpose

The goal of this portfolio is to show how technical problems are approached in a structured way: observe, measure, analyze, identify the root cause, and improve the process step by step.

## Project overview

- [Bluetooth RSSI Antenna Position Test](projects/bluetooth-rssi-antenna-position-test/)
- [X-ray USB Charger Inspection](projects/xray-usb-charger-inspection/)
- [USB Cable Testing](projects/usb-cable-testing/)
- [Laser Printer Root Cause Analysis](projects/laser-printer-root-cause-analysis/)
